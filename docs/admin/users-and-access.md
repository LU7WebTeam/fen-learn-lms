# Users and Access

## Roles

Current roles:

- `learner`
- `content_editor`
- `super_admin`

## User management

Admins can:

- browse staff and learners
- search accounts
- change roles
- view learner enrollment details
- edit user profile information
- suspend and unsuspend accounts

## Invitations

Staff invitation links are handled through token-based invitation routes.

This supports controlled onboarding of admin-side users.

## Profile access

Learners now have a visible `Profile` navigation link in the authenticated header.

The avatar dropdown still includes a profile shortcut as well.
