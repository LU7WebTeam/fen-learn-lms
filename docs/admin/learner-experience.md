# Learner Experience

## My Learning dashboard

The learner dashboard currently includes:

- welcoming hero section
- quick stats for enrolled, in-progress, and completed courses
- a continue-learning shortcut card
- in-progress course cards
- completed course cards

## Course player

Learners access lessons through the learning routes and can:

- start or resume lessons
- complete lessons
- submit quizzes
- view PDFs
- review completed courses

## Progress

Course progress is derived from completed lessons inside an enrollment.

This progress is used in:

- learner dashboard displays
- admin analytics
- course completion logic
- certificate readiness

## Learner profile

Learners can access profile management from the authenticated header.

Current editable fields include:

- name
- email
- avatar
- gender
- race / ethnicity
- state
- birthdate
- occupation
- organization / institution
