# Settings and Certificates

## Settings

Platform settings currently cover:

- platform name and tagline
- logo and favicon
- email testing
- custom font uploads

These values are shared into the frontend through Inertia when needed.

## Certificates

Certificates are configured at the course level.

Admins can manage:

- certificate enablement
- size and orientation
- background
- branding
- text fields
- signatory information
- font selection
- completion requirements

## Public certificate routes

Certificates can be viewed and downloaded publicly by UUID.
